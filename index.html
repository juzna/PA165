PA165 - Cards

This is a university project for PA165 course at FI MUNI. It's a web app where you can store pictures of business cards you receive.

Technologies

  • AppEngine
  • JPA (DataNucleus)
  • Spring MVC
  • Maven

Install

Just clone this repo and run mvn gae:run to get a local instance of AppEngine.
